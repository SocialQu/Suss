# Cluster Summary

## Algorithm Breakdown

1. Convert sentences to word embeddings.
2. Cluster sentences.
3. Dismiss clusters and sentences labeled as anomalies. 
4. For each cluster select the sentences that convey the most information.
5. Start building the minute.
